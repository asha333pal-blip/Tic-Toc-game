# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game developed using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Win detection logic
- Draw detection
- Restart/New Game option
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Player X starts the game.
2. Players take turns placing their marks (X or O) on the grid.
3. The first player to get three marks in a row, column, or diagonal wins.
4. If all cells are filled and no player wins, the game ends in a draw.

## Learning Outcomes
- DOM Manipulation
- Event Handling
- JavaScript Logic Building
- Basic Game Development
